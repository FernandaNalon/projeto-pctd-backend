# PCTD - Planejamento Coletivo de Trabalho Docente

## Objetivo

Sistema web para gerenciamento pedagógico e planejamento docente.

O objetivo é centralizar informações acadêmicas e administrativas, permitindo que docentes e coordenação acompanhem turmas, alunos, aulas, observações pedagógicas e unidades curriculares.

---

## Tecnologias utilizadas

### Backend

- NestJS
- TypeScript
- Prisma ORM
- MariaDB
- JWT
- bcrypt
- Swagger

### Frontend (futuro)

- TypeScript
